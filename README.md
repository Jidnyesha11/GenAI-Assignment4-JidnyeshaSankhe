## Description
This assignment demonstrates file handling in Python using a Jupyter Notebook (`.ipynb`). It covers creating, reading, appending, and processing text files, along with a mini project to generate a discount report.

---

## Tasks Completed

### Task 1 - Writing to a File
- Created a text file.
- Wrote sales data into the file.

### Task 2 - Reading from a File
- Read the contents of a text file.
- Displayed the file data using different file reading methods.

### Task 3 - Appending Data
- Appended new sales records to an existing file.
- Verified the updated file contents.

### Task 4 - Sales Summary
- Read sales data from a file.
- Calculated the total sales, highest sale, lowest sale, and average sale.

### Task 5 - Product Information File
- Accepted product names and prices from the user.
- Stored the data in a text file.
- Read and displayed the saved product information.

### Task 6 - File Exception Handling
- Checked whether a file exists before reading it.
- Displayed an appropriate error message if the file was not found.

### Task 7 - Mini Project: Export Discounted Prices
- Created a product price dictionary.
- Applied a user-defined discount percentage.
- Generated a `discount_report.txt` file containing:
  - Product Name
  - Original Price
  - Discounted Price
- Read and displayed the generated report.
- Included the optional summary showing:
  - Total Items
  - Average Discounted Price

## How to Execute

1. Open **task.ipynb** in Jupyter Notebook or VS Code.
2. Run the notebook cells from top to bottom.
3. Execute each task in sequence using **Run Cell** (`Shift + Enter`).
4. For tasks requiring user input, provide the requested values when prompted.
5. The generated text files (`sales_data.txt`, `product_data.txt`, and `discount_report.txt`) will be created in the same folder as the notebook.
